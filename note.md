# 一 容器的属性:
 - flex-direction　　　　设置主轴方向,即是X还是Y位主轴
 - flex-wrap　　　　　　　是否换行
 - flex-flow                
 - justify-content　　　　项目在主轴上的对齐方式 
 - align-items　　　　　　项目在附轴上的对齐方式,单行有效
 - align-content　　　　　项目在附轴上的对齐方式,多行有效

# 二  项目的属性
 - order 　　　　　　　　　定义子项的排列顺序,默认值是0,越小越靠前
 - flex-grow
 - flex-shrink
 - flex-basis
 - flex　　　　　　　　　　分配子项目的剩余空间
 - align-self             控制子项在侧轴的排列方式
 - 引入的文件路径建议用相对路径
